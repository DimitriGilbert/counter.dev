# Counter Frontend

React frontend for Counter built with TanStack Router, Vite, Tailwind CSS, and shadcn/ui.

## Development

```bash
pnpm install
pnpm dev
```

The dev server runs on port `3000`.

## Production Build

```bash
pnpm build
```

The static build is written to `dist/`.

The Docker image builds this app and serves it from the Go backend at `/app-next/`.
The Vite base path and TanStack Router base path are both configured for that mount point.

## Login State

The SPA reads `/api/session` with the existing Counter session cookie. Unauthenticated users see preview mode; authenticated users get their user id and preferred site links from the Go backend.

## Validation

```bash
pnpm exec tsc --noEmit
pnpm test
pnpm build
```

## Routing

Routes live in `src/routes/` and are generated by the TanStack Router Vite plugin.

## UI

shadcn/ui is configured in `components.json`. Installed components live in `src/components/ui/`.
